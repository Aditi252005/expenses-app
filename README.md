💰 FinTrack – Expense Tracker Web App

FinTrack is a full-stack expense tracking application that helps users manage their finances by recording income and expenses, categorizing transactions, and tracking their overall balance in real time.

---

🚀 Features 

🔐 User Authentication – Secure signup and login system for managing personal financial data

💵 Income & Expense Tracking – Users can add, edit, and delete transactions easily

📊 Real-Time Balance Calculation – Automatically updates the user's net balance based on income and expenses

🗂 Transaction Categorization – Organize expenses into different categories for better financial insights

📱 Responsive UI – Clean and responsive interface for seamless usage across devices

---

🛠 Tech Stack

Frontend- React.js, HTML, CSS

Backend- Node.js, Express.js

Database- MongoDB

Other Tools- JWT Authentication, RESTful APIs
